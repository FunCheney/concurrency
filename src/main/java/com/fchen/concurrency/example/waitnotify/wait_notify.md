### 线程间通信

&ensp;&ensp;线程拥有自己的栈空间，如若各个线程之前不能通信，其实对于多线程来说是没有意义的，因为只有各个线程之间相互协作，才能发挥多和处理器的最大价值。在前面Java内存模型的学习，也知道，java线程会将主内存的数据拷贝一份到自己的本地内存中，而线程之间通信的原理就是通过主内存来完成数据的共享。
